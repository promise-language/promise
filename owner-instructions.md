Before considering a task completed, it is mendatory to do the following:

** Tests **
Tests in main branch must pass at all times. Make sure when tasks are completed, all tests pass after you rebase the changes and before merging the task back to the main branch.

** Lint **
Before merging changes to the main branch, always run lint and fix any lint errors.

** Format **
Before merging changes to the main branch, always run code auto format to minimize formatting noise.

** Review **
Before merging changes do an meticulouse code review and fix issues
